**Project Title**

Project Title: MOODY BOT 


**📌 Project Overview**
The project is a line-following emotion robot built using an ESP32. It is designed to move along a black line using an IR sensor and motors, while also detecting colors with a TCS3200 color sensor and showing matching emotions on an OLED display. The system combines movement, sensing, and visual expression to create an interactive robot that reacts to its environment in a simple but engaging way.
The main idea of the project is to make a robot that does more than just follow a path. Along with navigation, it also responds emotionally based on the color it detects. This gives the robot a more lively and human-like behavior, which makes the project both functional and creative. The robot can be used as a demonstration of how sensors, embedded programming, and display systems can work together in a single embedded application.
